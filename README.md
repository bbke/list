##规则主页：http://code.minggo.tk##

###包含：###
      (1) ad.txt：整合iFL（贴吧清爽规则）、cjx和chinalist规则，以及我个人补充的一些规则;
      (2) ad2.txt：整合iFL（贴吧清爽规则）、xwhycadblock规则，以及我个人补充的一些规则;
      (3) gfwlist.txt：autoproxy中国规则的镜像，与官方同步更新;
      (4) proxy-list.txt：常用网站代理规则，替代繁杂、冗余的autoproxy中国规则;
      (5) pan.xpi：Firefox浏览器的扩展pan的修改版，提供广告过滤和自动代理功能;
      (6) hosts-android.txt：源自酷安网网友的github，保持同步更新，包含被墙站和去安卓大部分广告，
          并补充一些规则，加速网站访问;
      (7) hosts-pc.txt：根据hosts-android.txt精简，删除去安卓广告的部分;
      (8) YoukuAntiADs.uc.js：修改自jiayiming提供的YoukuAntiADs.uc.js脚本，修改为本站分流的播放器地址;
      (9) redirector.uc.js：提供重定向功能，已经包含google搜索结果禁止跳转、wiki繁转简、
          Google公共库>>360公共库、Google统计脚本的正常使用等规则。
