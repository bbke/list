##规则主页：http://code.minggo.tk##

###包含：###
      (1) ad.txt：整合iFL（贴吧清爽规则）、cjx和adblockchina规则，以及我个人定义的一些去广告规则。
      (2) gfwlist.txt：autoproxy中国规则的镜像，与官方同步更新。
      (3) proxy-list.txt：常用网站代理规则，替代庞大的autoproxy中国规则。
      (4) pan.xpi：Firefox浏览器的扩展pan的修改版，提供广告过滤和自动代理功能。
      (5) hosts-android.txt：源自酷安网网友的github，保持同步更新，包含被墙站和去安卓大部分广告，
          并补充一些规则，加速网站访问。
      (6) hosts-pc.txt：根据hosts-android.txt精简，删除去安卓广告的部分。
